# Не сегодня

Мистическое PWA, которое каждый день выдаёт персональную причину не работать.

## Возможности

- 365-дневный цикл астрологических и магических обоснований
- персональный локальный знак для каждого пользователя или устройства
- исходный прогноз и до трёх сохранённых перебросов в день
- интерактивный магический шар
- готовая справка для начальника, коллеги или заказчика
- копирование и системная отправка обоснования
- выбор даты в архиве знаков
- установка как приложение и полноценная офлайн-работа

Персонализация хранится только на устройстве в `localStorage`; аккаунт и сервер не требуются.

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Production-сборка

```bash
pnpm build
pnpm preview
```

Интерфейс рассчитан прежде всего на мобильный экран 390 × 844, но корректно открывается и в настольном браузере.
